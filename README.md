## Compiling
To compile Beam.Net yourself, you will need either of the following:

##### Visual Studio
- [Visual Studio 2017](https://www.microsoft.com/net/core#windowsvs2017)

##### Command Line Tools
- [.Net Core 1.1 SDK](https://www.microsoft.com/net/download/core)
